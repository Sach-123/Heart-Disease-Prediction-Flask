About this System...

Heart Disease Prediction System is a simple machine learning model which classify whether the patient have heart disease or not based on patient's data. The system uses logistic regression for calculating the result. The model produced an approx. accuracy of 83% for training data and 80% for testing data.
The data set used has around 300 records. You can check the dataset by clicking here.

For Building Ml model following module of python was used :
  • Pandas
  • Numpy
  • Sklearn

    • Deployment is done with the help of flask and for styling of the webpage tailwind CSS is used.
